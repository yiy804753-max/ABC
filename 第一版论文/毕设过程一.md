# 毕设过程一

## 第1章 概述

### 1.1 课题背景

娱乐圈和大众文化联系很紧，演员、歌手、导演、主持人、运动员等公众人物一直处在大众视野中。一旦出现违法犯罪事件，很容易在短时间内引起讨论。这些事件看似是个人问题，但也和行业管理、法律意识以及舆论传播有关。

目前这类信息大多来自新闻报道、热搜话题和短视频解读，观众可以很快知道某个事件的经过，但很难看到整体情况。比如某位明星涉及偷税、涉毒、暴力或诈骗，大家可能只记住了单个事件，却不容易了解不同犯罪类型的占比，也看不出这些案例在时间、地区、性别和年龄上的分布。

互联网让信息传播更快，也让内容变得更零散。一个事件被反复推送和讨论后，观众往往停留在热点本身，缺少对整体案例的梳理。因此，本课题希望把这些分散信息整理起来，用更清楚的视觉方式呈现出来。

信息可视化能够把复杂资料转化成更直观的图形内容。对于娱乐圈违法犯罪案例来说，它不仅能呈现具体人物和事件，也能通过时间线、比例图表和分类图例展示其中的关系。

因此，本课题以“娱乐圈犯罪实录”为研究对象，对全球相关案例进行整理、分类和视觉呈现。设计重点不是放大个别事件，而是用相对客观的方式梳理信息，让观众更直观地了解这类现象的整体情况。

### 1.2 理论与实践意义

#### 1.2.1 理论意义

从理论上看，本课题主要是把信息可视化方法用到社会事件类内容中，尝试把原本零散的案例整理成更清楚的视觉结构。娱乐圈违法犯罪案例包含人物、时间、地区、类型、性别、年龄等信息，如果不先筛选和分类，内容很容易变得杂乱，阅读时也缺少顺序。

本课题采用全球总图和地区分图的形式，把整体数据和具体案例结合起来。全球总图用来呈现总体情况，地区分图则通过S-Curve蛇形时间线梳理人物案例，让观众可以按照时间顺序阅读，先看整体，再看细节。

在表现方式上，课题加入了人物插画。相比单纯使用图表，插画更容易让观众记住人物；相比直接使用照片，插画又能让画面风格更统一，使整套作品看起来更完整。

信息轮盘测试也对前期调整起到了一定作用。通过测试，可以检查数据是否准确、层级是否清楚、版式是否能放下足够信息。由此也能看出，信息可视化不只是最后的画面呈现，还包括前期整理、反复验证和修改。

#### 1.2.2 实践意义

从实践部分来看，本课题最终完成了一套系列信息图，包括一张全球总图和五张地区分图，同时制作了网页展示页面，方便展览展示和线上浏览。

作品把分散的娱乐圈违法犯罪案例重新整理。观众可以先通过全球总图了解整体情况，再看地区分图中的人物和事件，最后结合比例图表查看性别、时间、年龄等数据，阅读过程会更清楚。

在设计方法上，作品结合了数据整理、人物插画、时间线、比例图表、高饱和色彩和网页展示等内容，尝试用信息图的方式表现社会议题。

从毕业设计整体来看，课题包含前期调研、案例分析、草图推演、信息轮盘测试、最终设计、展板制作、实物输出和网页延展，流程比较完整，也能体现从调研到成品展示的设计过程。

### 1.3 国内外发展概况及存在问题

#### 1.3.1 国外信息可视化设计发展概况

国外信息可视化设计发展较早，应用范围也比较广，常见于新闻传播、公共服务、商业分析、科学研究和艺术表达等领域。在数据新闻中，信息图经常被用来解释社会事件、选举数据、经济变化、公共卫生和环境议题等内容。

例如卡洛斯·阿劳霍、乔·罗、马修·郑、伊迪丝·林、卡利兹·李、刘嘉坤、亨利·黄、布莱恩·王共同发表的《立法会选举》（2025），就以香港立法会选举为主题。作品用橙色、绿色、紫色区分地理选区、功能界别和选举委员会界别，同时展示当选人头像、得票数、姓名、年龄和所属政党等信息。整体采用密集网格排版，信息量较大，但层级比较清楚，底部还补充了投票率变化、政党说明和新闻背景，具有较强的新闻信息图特点。通过色块、地图、时间轴、比例图和关系图等形式，设计者可以把复杂数据转化为更容易阅读和理解的视觉内容。

国外很多信息图并不只是展示数字，而是把数据和故事线放在一起处理。比如用时间线梳理事件变化，用头像、标签和简短说明呈现人物个例，让观众在看数据的同时，也能对应到具体的人和事。这一点对本课题有参考意义，因为娱乐圈违法犯罪案例既要看统计结果，也要保留人物和事件本身。

随着数字媒体发展，国外信息可视化也从静态图表延伸到网页、互动界面和动态展示。观众可以通过点击、筛选、滚动等方式查看内容，阅读方式更灵活。

本课题虽然以系列信息图为主，但在网页展示部分也参考了一部分这种线上延展形式。

#### 1.3.2 国内信息可视化设计发展概况

国内信息可视化设计近年来发展较快，已广泛应用于新闻媒体、政府宣传、文化传播、教育科普和商业展示等领域。在重大节日、社会事件、文化遗产和城市数据展示中，信息图常被用来提高信息传播效率。相比传统文字报道，信息图更便于社交媒体传播，也更符合当下观众快速浏览和视觉化阅读的习惯。

在视觉传达设计教育中，信息图形设计也逐渐成为常见研究方向。许多毕业设计会围绕文化、历史、社会现象或公共议题展开，通过数据整理和图形转译形成系列作品。例如刘家权的《残酷的竞争》，以“鼠年”为主题，采用红金配色和传统纹样营造春节氛围，介绍生肖年份、五行属性、幸运元素、身体特征、生肖配对和属鼠名人等内容。作品信息丰富，中心插画突出，属于较典型的文化专题信息图。此类案例说明，信息图形设计能够让复杂内容形成更清晰的展示结构。

但在社会议题类信息图设计中，也存在一些问题。比如有的作品过分强调画面装饰，反而忽视了数据是否准确；有的为了吸引眼球，使用夸张或情绪化的表达，影响了信息的客观性；还有的在信息量较大时，阅读顺序不够清楚，观众不容易抓住重点。因此，本课题在设计时需要注意表达的中立性，合理安排信息层级，并处理好图例、数据和阅读顺序之间的关系。

#### 1.3.3 社会议题类信息图形设计存在的问题

社会议题类信息图多涉及公共事件、法律问题和社会现象，内容复杂，也容易受舆论情绪影响。如果过分追求视觉冲击，主题容易被夸大；如果只强调数据，又会削弱具体人物和事件。因此，这类设计需要平衡数据和叙事。

娱乐圈违法犯罪案例较为特殊，相关人物关注度高，事件又涉及法律和社会规范，不适合用猎奇或戏剧化方式呈现。本课题因此采用中立、清晰的表达方式，重点放在信息整理和数据关系呈现上。

同时，本课题案例信息较多，包含姓名、职业、地区、时间、犯罪类型、年龄、性别等内容。全部放进画面会显得拥挤，观众也难抓住重点。因此，设计时需要分清主次：先看标题、地区和总数，再展示人物、时间和犯罪类型，最后用比例图表补充统计结果。

### 1.4 课题研究的主要内容

本课题围绕“娱乐圈犯罪实录”开展信息可视化设计，主要从以下几个方面进行。

第一，整理全球代表国家中娱乐圈违法犯罪案例，提取人物、地区、时间、犯罪类型、性别、年龄等内容，为后续设计提供资料依据。整理过程中，对信息来源、分类方式和统计结果进行核对，保证图表内容相对准确。

第二，查找并分析相关信息图案例，重点参考时间线、人物个例和系列化版式的表现方式，了解其在人物排列、版面统一、图例说明和色彩区分上的处理方法。

第三，确定“1+5”的作品结构，即一张全球个例总图和五张地区个例分图。全球总图用来呈现整体情况，地区分图则展示具体人物案例，使观众能够从整体逐步看到局部内容。

第四，统一作品的视觉语言。设计中用高饱和色彩区分犯罪类型，用人物插画增强识别度，并通过S-Curve蛇形时间线梳理地区案例，使六张作品在字体、图例、图表和版式上保持一致。

第五，完成最终设计及展示延展，包括效果图、展板、实物输出和网页展示页面，使作品能够用于现场展示，也方便线上浏览。
