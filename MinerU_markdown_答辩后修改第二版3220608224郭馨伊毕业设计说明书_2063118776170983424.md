# Abstract

This project focuses on the theme of “Criminal Records in the Entertainment Industry”. It responds to the fragmented dissemination of criminal cases involving public figures in the online media environment. The project will collect public reports and related literature, extract basic fields such as name, region, time, crime category, gender and age, and then clean, classify and analyze the data. The final design will adopt a “1+5” structure, including one global overview infographic and five regional infographics. The regional works will use an S-Curve timeline to organize individual cases. High-saturation color legends, portrait illustrations, proportional charts and visual symbols will be used to present crime categories, time distribution and basic statistical relationships. The project aims to transform scattered social event information into a readable and systematic visual series. It will also explore the value of information visualization in legal publicity, public communication and visual communication design education.

# 引 言

信息时代改变了人们获取和传播信息的方式。网络媒体、社交平台和短视频平台让大量社会事件更快进入公众视野，也使信息呈现出高频率、碎片化和多渠道传播的特点。娱乐圈作为关注度较高的社会文化领域，其从业人员的违法犯罪事件往往会在短时间内引发讨论。这类事件传播快、话题性强，但在传播过程中容易被简化为个体事件、网络热点或娱乐谈资，使公众难以系统了解其类型分布、时间规律和社会关系。

“娱乐圈犯罪实录”这一选题并不是对个别人物事件的猎奇呈现，而是从信息整理和视觉表达的角度，对相关案例进行归纳、分类和重组。违法犯罪案例具有一定的社会属性和公共讨论价值，但如果只通过文字新闻或表格呈现，观众很难快速形成整体认识。信息可视化设计则可以借助图形、色彩、比例、时间线和空间布局等方式，将复杂内容转化为更直观的视觉信息，帮助观众在较短时间内理解数据之间的关系。

本课题以全球娱乐圈违法犯罪案例为基础，通过资料收集、数据分类、比例统计、人物插画和系列版式设计，构建“1+5”结构的信息图作品。所谓“1+5”，即一张全球个例总图和五张地区个例分图。全球总图主要承担整体概览和图例说明功能，五张地区分图则以S-Curve蛇形时间线为主要形式，展示不同地区的犯罪人员、犯罪时间与犯罪类型。通过总分结合的方式，作品既呈现全球范围内的整体信息，也对各地区个例进行更细致的视觉表达。

在视觉表现上，本课题以 A3纸张作为主要输出规格，并用高饱和色彩区分不同犯罪类型。人物形象采用插画化处理，既提高个例识别度，也让整体风格更加统一。在信息层级上，作品将标题、地区和总数作为第一层信息；犯罪人员、犯罪时间和犯罪类型作为第二层信息；男女比例、时间比例、时间段比例、年龄比例和图例演示作为第三层信息。通过由浅入深的阅读路径，观众可以先了解整体内容，再查看人物个例，最后通过比例图表理解数据特征。

本论文围绕该课题的研究背景、设计构思、作品分析和制作过程展开。第一章主要说明课题背景、意义和研究内容；第二章分析设计调研、案例研究和视觉语言系统；第三章重点解读草图推演、信息轮盘测试、全球总图和五张地区分图的设计特点；第四章介绍效果图制作、展板制作、实物输出和网页展示延展。通过完整的设计研究与实践，本课题希望探索社会事件类信息如何通过视觉设计转化为清晰、系统且具有传播价值的系列信息图形作品。

# 第1 章 背景与意义

# 1.1 课题背景

近年来，法治建设、网络治理和公众信息传播环境不断发展。中共中央印发的《法治社会建设实施纲要（2020—2025年）》提出，要推动全社会增强法治观念，提高社会治理法治化水平。中央宣传部、司法部关于“八五”普法的相关规划也强调，要增强法治宣传教育的针对性和实效性。由此可见，法治信息的传播不应只停留在条文层面，也需要通过更适合大众阅读的方式进入公共传播空间。

与此同时，网络空间已经成为社会事件传播的重要场域。《中华人民共和国网络安全法》强调维护网络空间秩序和社会公共利益。《中华人民共和国个人信息保护法》也对个人信息处理和权益保护提出要求。这些法律背景提醒本课题，在处理公众人物相关案例时，应当使用公开信息，避免扩散无关隐私，避免用视觉设计制造二次伤害。

娱乐圈违法犯罪事件具有较高传播度。演员、歌手、导演、主持人、运动员等公众人物处在大众文化的中心位置。当相关违法犯罪事件出现时，网络平台会快速放大讨论。公众可以很快知道事件本身，却不容易看到整体分布。例如，经济类、毒品类、暴力类和性犯罪类案件在不同地区和时间段中如何出现，相关人物年龄和性别是否呈现某些比例特征，这些问题都需要通过整理之后才能被看见。

因此，本课题将“娱乐圈违法犯罪实录”作为研究对象，并将其放在法治传播和视觉传达设计的交叉位置来考察。课题不是简单收集事件，也不是制作娱乐化榜单，而是希望通过信息可视化方法，将分散资料转化为可阅读、可比较、可展示的视觉内容。

# 1.2 理论意义

信息可视化强调将抽象数据转化为可感知的视觉形式。它不仅处理数字，也处理信息之间的关系。社会事件类内容往往包含时间、地点、人物、类型和结果等多维信息。如果缺少组织方式，信息就会停留在堆叠状态。通过视觉编码、层级划分和叙事路径设计，复杂信息可以被重新组织。

本课题的理论意义首先体现在社会议题与信息可视化方法的结合。娱乐圈违法犯罪案例并不是传统意义上的纯数据对象，而是由新闻文本、法律事实和公众讨论共同构成的复合信息。课题将从这些材料中提取可视化字段，并通过时间线、图例、比例图表和人物插画建立对应关系。这一过程可以丰富社会事件类信息图的表达方式。

其次，本课题将探索“总分结合”的视觉叙事结构。全球总图承担总体概览和图例说明功能，地区分图承担个例展开功能。观众先获得整体印象，再进入具体案例，最后回到统计关系。这种结构能够避免单张图信息过载，也能让系列作品保持统一。

再次，课题将通过S-Curve蛇形时间线处理人物案例。相比直线时间轴，蛇形结构更适合在A3 版面中安排较多节点。它既能保留时间顺序，也能给人物插画和说明文字留出空间。因此，本课题对时间线叙事、人物节点设计和信息层级组织具有一定探索意义。。

# 1.3 实践意义

本课题的实践意义主要体现在三个方面。第一，它将为社会议题类信息图设计提供一个可执行的流程。该流程包括资料收集、字段拆分、数据清洗、分类统计、草图推演、信息轮盘测试、视觉系统建立和最终输出。通过这些步骤，设计作品将不只是视觉包装，而是建立在信息整理基础上的综合表达。

第二，作品将服务于线下展览和线上传播。线下部分以 A3 信息图、展板、实物衍生品和亚克力展示为主，适合毕业设计展陈；线上部分将通过网页展示项目介绍、设计过程和六张最终信息图，便于观众继续浏览。不同媒介共同构成传播路径，使作品具有更完整的展示方式。

第三，作品将发挥一定的法治宣传和公众阅读作用。公众人物违法犯罪事件本身具有警示意义。通过中立、清晰的视觉方式呈现相关案例，观众可以从个别事件转向整体观察，从情绪讨论转向类型、时间和比例关系的理解。该设计将有助于提升社会议题的可读性，也能为法治宣传提供视觉化参考。

# 1.4 国内外信息可视化设计发展概况

# 1.4.1 国内信息可视化设计发展概况

国内信息可视化设计近年来发展较快，应用范围不断扩大。在新闻报道、公共服务、文化传播、教育科普和商业展示中，信息图已经成为常见的表达方式。相比单纯文字，信息图更适合在社交媒体和展览空间中传播。它可以用图形、图表、地图和时间线压缩信息，使观众在较短时间内理解内容重点。

不过，社会议题类信息图仍存在一些问题。部分作品过度追求视觉冲击，容易削弱数据准确性；部分作品信息层级不清，导致观众难以判断阅读顺序；还有一些作品在表达法律和公共事件时语气过重，容易产生情绪化倾向。因此，本课题将在设计中强调中立表达、层级组织和图例规范。它将通过“1+5”系列结构、S-Curve时间线和统一标志系统，把娱乐圈违法犯罪案例转化为更清楚的视觉资料。

# 1.5 课题研究的主要内容

本课题将完成一套以“娱乐圈违法犯罪实录”为主题的信息可视化作品。作品包括一张全球个例总图和五张地区个例分图，形成“1+5”的系列结构。五张地区分图分别对应亚洲之中国、非洲之尼日利亚、美洲之美国与巴西、大洋洲之澳大利亚、欧洲之法国。

作品将以A3 纸张作为主要输出尺寸。主视觉内容包括人物插画、S-Curve蛇形时间线、犯罪类型图例、比例统计图和地域装饰元素。后续展示将包含展板、实物延展、网页展示和相关衍生设计。

字体设计方面，标题字体将承担主题识别功能，正文字体将服务人物姓名、犯罪时间、犯罪类型和图表说明。字体选择将以清晰、稳定、易读为原则。

标志体系方面，作品将建立人物信息类、时间轴与节点类、地图与地域信息类、版式辅助类等符号。装饰元素将结合不同地区的地标、建筑和文化符号，但其作用应服务信息识别，不应干扰主体内容。

色彩体系方面，作品将使用高饱和色彩区分犯罪类型，并在六张信息图中保持一致。观众可以通过图例快速理解色彩含义，再回到人物节点中识别案例类别。

# 第 2章 设计方案与构思

# 2.1 总体方案

本课题的总体方案是以“娱乐圈违法犯罪实录”为叙述主题，围绕公众人物违法犯罪案例的收集、整理和视觉呈现展开。设计对象不是单个新闻事件，而是一组具有公共讨论价值的案例资料。课题将通过信息可视化方法，把人物、时间、地区、犯罪类型、性别、年龄等内容重新组织，使其形成可阅读的系列信息图。

在执行步骤上，课题将先进行资料收集。资料来源以公开报道、文献资料和可核验信息为主。收集后，将把长文本拆分为统一字段，形成可整理的数据表。字段包括人物姓名、所属地区、职业类型、犯罪时间、犯罪类型、性别、年龄和法律结果等。

其次，课题将进行数据清理。清理内容包括统一人物名称、统一犯罪类型、删除重复信息、核对时间和地区，并对无法确认的信息作谨慎处理。对于不同国家和地区法律表述不完全一致的问题，设计中将以视觉分类为主要原则，将犯罪类型归纳为经济类、暴力类、毒品类、性犯罪类、诈骗赌博类、造假类等。

再次，课题将进行数据归纳和统计。基础数据集用于人物节点制作，分项数据集用于男女比例、犯罪时间比例、时间段比例和年龄比例统计。这样可以保证人物个例与统计图表互相对应。

最后，课题将进入视觉转化阶段。全球总图将承担入口作用，地区分图将承担具体叙事作用。人物插画、色彩图例、时间线节点、比例图表和地域装饰将共同构成视觉系统。整体设计将遵循信息图形设计中准确、清晰、层级明确的原则，避免只追求装饰效果。

# 2.2 设计框架

本课题的设计框架分为四个层面。第一层是数据框架。该层面解决“有什么信息”的问题。通过资料收集和字段整理，作品将获得人物、地区、时间和类型等基础内容。

第二层是叙事框架。该层面解决“先看什么、后看什么”的问题。全球总图将提供整体概览，地区分图将按时间顺序展开个例。观众可以从总图进入，再根据地区阅读人物案例。

第三层是视觉框架。该层面解决“信息如何被看见”的问题。人物插画用于建立识别点，S-Curve时间线用于组织时间顺序，高饱和色彩用于区分犯罪类型，比例图表用于补充统计关系。

第四层是传播框架。该层面解决“作品如何被展示”的问题。线下作品将用于毕业设计展览和实物陈列，线上网页将用于完整呈现项目介绍、过程和最终成果。通过这四个层面，作品将达到信息清楚、风格统一、阅读有序和传播便利的效果。

# 2.3 案例调研

案例调研主要围绕时间线信息图、人物个例信息图和系列化版式展开。时间线类作品适合呈现事件顺序。它能让观众沿着时间路径阅读，理解事件在不同阶段的变化。本课题中的案例都具有明确年份，因此时间线将成为重要结构。

人物个例类信息图通常以头像、插画、姓名和标签形成视觉单元。与单纯数据图表相比，人物个例能让观众快速知道“是谁”和“发生了什么”。本课题中的案例与具体人物相关，因此将采用人物插画作为视觉入口。这样既能提高识别度，也能避免真实照片风格不统一的问题。

系列化版式案例对本课题也有参考价值。六张作品需要保持统一的标题系统、图例规则、字体关系和色彩标准。如果每张图风格差异过大，观众就难以把它们看成一套完整作品。因此，本课题将在纸张尺寸、图例位置、色彩规则和底部统计图等方面形成统一标准。

# 2.4 设计定位

本课题定位为公益性法制宣传类信息可视化设计。设计不以猎奇娱乐为目的，也不对个别人物和事件作主观评价，而是通过资料整理、案例分类和视觉转化，将分散的娱乐圈违法犯罪案例整理成清晰、可读的视觉信息。这样可以让观众在观看作品时，不只关注单一事件，也能看到不同案例之间的时间关系、类型差异和统计特征。

从内容定位来看，作品主要围绕人物、地区、时间、犯罪类型和比例统计展开。娱乐圈从业者具有较高社会关注度，其违法犯罪事件容易引发公众讨论。因此，本设计希望借助信息可视化的方式，引导观众关注事件背后的法律边界、公众责任和社会规范，使作品具有一定的警示和教育作用。

从传播定位来看，作品将分为线上传播和线下展示两个方向。线上方面，作品可延展为网页展示、长图海报、系列推文或短图文内容，用于小红书、微信公众号、微博、抖音图文、网页端等平台传播，便于观众在移动端浏览和转发。线下方面，作品可用于毕业设计展览、校园法制宣传、印刷海报展示、展板陈列和实物延展展示等场景，使观众能够在现场完整阅读系列信息图。

因此，本课题将在中立、清晰、克制的视觉语言基础上展开设计。画面通过高饱和色彩区分犯罪类型，通过人物插画增强识别度，并用S-Curve蛇形时间线组织案例顺序。整体表达将突出信息的准确性、阅读的顺畅性和公益性法制宣传的警示作用。

# 第 3 章 设计制作流程

# 3.1叙事框架搭建

叙事框架是本课题从资料走向视觉的关键步骤。前期草图曾尝试地图分布式、矩阵排列式和轮盘式结构。地图分布式能突出地区关系，但不适合承载大量人物节点；矩阵排列式秩序感较强，但时间顺序不明显；轮盘式视觉中心突出，但人物案例过多时阅读方向容易混乱。

经过比较，课题最终将采用“全球总图+地区分图”的叙事方式。全球总图先建立整体印象，包括主题、地区、图例和基础统计。地区分图再以S-Curve蛇形时间线展开人物个例。这样的叙事方式能让观众从整体进入局部，也能减少单张图的信息压力。

在地区分图中，S-Curve蛇形时间线将作为主要版式骨架。人物节点沿曲线排列，时间信息与节点对应，犯罪类型通过色彩呈现。观众顺着曲线阅读，就能从较早案例看到近年案例。

# 3.2数据收集

数据收集阶段以公开信息为基础。收集内容包括新闻报道、公开文献、法律报道和相关资料。为了保证后续设计能够使用，资料在收集时不只记录事件经过，还要提取可视化所需字段。

本课题将重点记录人物姓名、国家或地区、职业身份、犯罪时间、犯罪类型、性别、年龄和法律结果等内容。其中，人物和时间用于时间线节点，犯罪类型用于色彩图例，性别和年龄用于比例图表，地区用于系列分图。

在收集过程中，需要注意信息边界。涉及个人隐私、未经证实的网络传言和与主题无关的内容，不应进入最终数据集。对于不同来源存在差异的信息，应以更可靠的公开资料为准，并在表格中进行标注。

# 3.3数据清洗

数据清洗是为了让资料能够进入设计系统。原始资料通常来自不同平台，表述方式并不一致。例如，同一类事件可能被写成不同罪名，不同报道对时间和结果的描述也可能有差异。如果不清洗，后续图例和统计图就会出现混乱。

本课题的数据清洗主要包括四个步骤。第一，统一字段名称。所有案例都按照同一表头整理，避免后期遗漏。第二，统一犯罪类型。将相近事件归入同一视觉类别，使图例数量保持可控。第三，核对时间、地区和人物信息。对于重复案例进行合并，对无法确认的内容进行剔除或备注。第四，生成统计字段。性别、年龄、时间段和犯罪类型等内容将被转换为可统计数据。

清洗后的数据将同时服务人物节点和比例图表。人物节点保留最关键的信息，比例图表则使用分项统计结果。这样可以避免每个节点承载过多文字，也能保持画面阅读清

晰。

# 3.4数据集制作

本课题的数据集将分为基础数据集和分项数据集。基础数据集用于记录每一个人物案例，是全部信息图的底层资料。分项数据集则根据设计需要生成，主要包括犯罪类型统计、男女比例统计、犯罪时间比例统计、时间段比例统计和年龄比例统计。

基础数据集将直接影响人物节点制作。每个节点将对应一个人物案例，包含姓名、时间和犯罪类型等主要内容。为了便于阅读，节点信息不宜过长。涉案金额、法律结果等内容可根据版面需要进行选择性呈现。

分项数据集将用于底部比例图表。男女比例可以帮助观众了解案例中的性别分布；犯罪时间比例和时间段比例可以呈现事件在不同阶段的集中情况；年龄比例可以显示案例人物的年龄结构。统计图表属于第三层信息，应当清楚但不抢占主体。

在工具使用上，数据整理阶段可借助表格软件完成字段录入、筛选和数据透视。视觉设计阶段可使用设计软件绘制人物插画、时间线、图例和版式。数据表与设计稿之间需要反复核对，保证最终图表与原始数据一致。

# 3.5 信息层级与阅读路径设计

本课题的信息阅读路径分为三层。

第一层是标题、地区和总数，作为画面的主要入口，让观众先判断这张图讲什么、对应哪个地区、包含多少案例。

第二层是犯罪人员、犯罪时间和犯罪类型，也是地区分图的主要内容。人物节点沿S-Curve蛇形时间线排列，观众可以顺着时间阅读案例，并通过高饱和色彩识别犯罪类型。

第三层是男女比例、犯罪时间比例、时间段比例、年龄比例和图例说明，用来补充整体数据特征，帮助观众在看完个例后进一步理解统计关系。

这种“标题总览—人物个例—比例统计”的层级结构，让作品既有阅读顺序，也保留了数据分析功能，观众可以快速浏览，也可以深入查看。

# 4.1.1 标题字体设计

标题字体主要用于突出主题。每张信息图都需要通过标题让观众快速知道内容，比如全球总图、中国地区分图、美国地区分图等。因此，标题要有一定视觉重量，方便第一时间识别主题。

标题设计不宜过度装饰，字体应清晰、稳定，避免影响阅读。六张作品在标题字号、位置和排版方式上保持统一，也能增强整体系列感。

# 4.4 版式设计

版式设计将围绕 A3 纸张展开。A3尺寸既能承载人物插画和统计图表，也适合毕业设计展览和后期打印。全球总图和地区分图虽然功能不同，但应保持统一的视觉秩序。

全球总图将作为系列入口。它需要呈现主题名称、地区关系、犯罪类型图例和基础统计内容。地区分图则以S-Curve蛇形时间线为主体，人物节点沿曲线排布，底部或边缘区域用于比例统计图和图例说明。

在阅读路径上，作品将形成三层结构。第一层是标题、地区和总数，帮助观众快速判断主题。第二层是犯罪人员、犯罪时间和犯罪类型，是主要阅读内容。第三层是男女比例、犯罪时间比例、时间段比例、年龄比例和图例说明，用来补充数据关系。

# 4.5.1 数据整理与分类

效果图制作首先从数据整理开始。收集到的原始资料需要整理成设计表格，内容包括犯罪人员、地区、犯罪时间、犯罪类型、性别、年龄等，这些分类会直接影响后面的图例、色彩和比例图表。

整理时要统一犯罪类型名称，避免同类事件出现不同叫法，导致图例变复杂。同时，性别、年龄和时间等统计信息也需要反复核对，保证后续比例图表准确。

同时，画面缺少明确的标题、图例和阅读引导，信息层级不够清楚，容易变成人物罗列，而不是完整的信息可视化。经过前期对比，本课题更需要一种既能承载人物个例，又能呈现时间顺序，并适合 A3 展示的结构，因此时间线逐渐成为主要方向。

# 4.5.3 S-Curve 蛇形时间线方案确立

![image](https://cdn-mineru.openxlab.org.cn/result/2026-06-06/5632bf01-ef65-498e-a0a2-3cb84a80e674/38c9b9afd92af6ecac5ce559fca32a4da8ad071d22cdb52be99125cdb5ffdaaf.jpg)



图 4-10 四稿草图



来源：本人制作


经过多轮对比后，地区分图最终采用S-Curve蛇形时间线。它可以把人物案例按时间排开，在A3 画面里形成一条连续的阅读路线。观众顺着曲线阅读，就能从早期案例看到近年案例，了解事件在不同时间段的分布。

相比普通直线时间轴，蛇形结构更适合放置较多节点。直线时间轴案例一多就容易拥挤，而蛇形曲线可以通过转折分散空间，让人物插画、姓名、时间和类型标识都有位置，画面也不会太死板。

在最终地区分图中，S-Curve不只是装饰，而是主要版式骨架。人物节点沿曲线展开，时间信息与节点对应，犯罪类型用高饱和色彩区分，使人物、时间和分类信息能更清楚地结合在一起。

# 4.5.4 人物插画与时间线节点制作

人物插画是效果图里比较关键的一步。根据整理好的案例资料，先绘制统一风格的人物插画，再把姓名、犯罪时间和犯罪类型一起组合成节点。

时间线节点排版时，要注意位置、大小和间距。节点沿 S-Curve 蛇形曲线展开，既要看得出时间顺序，也要避免互相遮挡。案例多的地区可以适当压缩间距，案例少的地区则调整曲线节奏，让画面看起来更均衡。

# 4.5.5 比例统计图制作

比例统计图包括男女比例、犯罪时间比例、时间段比例和年龄比例。制作前先根据表格计算数据，再转化成适合画面展示的图表形式。图表的字体、线条和色彩需要与整体风格保持一致。

这部分属于第三层信息，主要起辅助说明作用，不应抢过标题和人物时间线的重点。图表需要清楚好读，但视觉重量要适当降低，用来帮助观众理解整体数据。

# 4.5.6 六张系列信息图整合

全球总图、五张地区分图、人物插画、时间线节点和比例图表完成后，需要进行整体整合。重点检查纸张尺寸、标题系统、图例规则、色彩和字体是否统一。

六张作品内容不同，但要保持同一系列的感觉。全球总图作为入口，地区分图作为展开，视觉语言需要一致。整合时还要检查文字是否清楚、节点是否拥挤、图例是否完整，以及比例图表数据是否准确。

# 结论与展望

本课题以“娱乐圈犯罪实录”为主题，对全球娱乐圈违法犯罪案例进行整理、分类和视觉呈现。最终作品由一张全球个例总图和五张S-Curve蛇形时间线地区分图组成，把原本分散在新闻和网络资料中的案例，整理成一套结构较清楚的系列信息图。

在研究过程中，课题先完成资料收集和数据整理，再通过相关信息图案例调研确定设计方向。前期也尝试了不同的信息组织方式，并利用信息轮盘测试检查数据是否准确、版式是否能承载足够内容。最终选择S-Curve蛇形时间线作为地区分图的主要结构，用高饱和色彩区分犯罪类型，并通过人物插画增强个例识别度。

从最终效果来看，作品的信息层级较清晰：标题、地区和总数作为第一层，帮助观众快速了解主题；犯罪人员、犯罪时间和犯罪类型作为第二层，构成主要阅读内容；男女比例、时间比例、时间段比例、年龄比例和图例说明作为第三层，用来补充数据分析。这样的安排既能呈现具体人物案例，也能展示整体数据特点。

本课题也说明，社会事件类信息可以通过视觉设计变得更有条理。娱乐圈违法犯罪案例原本比较零散，经过可视化整理后，观众能更直观地看到其中的分布关系。作品在表达上尽量保持中立，不做夸张化和娱乐化处理，把重点放在信息本身。

当然，课题还有继续完善的空间。后续可以扩充更多国家和年份案例，也可以加入职业类型、法律结果、事件影响等统计维度。网页部分还可以进一步做成交互式展示，让观众按地区、时间或犯罪类型筛选案例，并延展为动态信息图或线上展览。

总体来说，本课题完成了从资料收集、数据整理、方案测试、视觉设计到展示延展的完整过程。通过这次实践，也进一步认识到信息图形设计并不只是把数据做成图表，更重要的是把复杂信息重新梳理、组织和表达出来。
